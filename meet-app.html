Meet App

The objective for this app was to build a serverless PWA with React, focusing on TDD. This app uses the Google Calendar API to fetch events. Users can view visualizations of the genre of events available and where they are. Users can search for events by city.

Technologies Used

  • React
  • Redux
  • Recharts
  • Google Calendar API
  • AWS Lambda
  • Jest
  • Puppeteer
